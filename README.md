# Para Yoangelis

Sorpresa web móvil para Android. Funciona como sitio estático y como PWA básica.

## Cómo abrir localmente

1. Descarga el ZIP y descomprímelo.
2. Abre `index.html` en Chrome desde una computadora o súbelo directamente a un hosting estático.

Para probar el service worker/PWA es mejor subirlo a internet, porque muchos navegadores bloquean esas funciones desde archivos locales.

## Cómo subir rápido

### Netlify
Arrastra la carpeta completa `yoangelis_app` a Netlify Drop.

### GitHub Pages
Sube todos los archivos a un repositorio y activa Pages desde Settings > Pages.

### Vercel
Crea un proyecto nuevo e importa la carpeta como sitio estático.

## Personalización rápida

- Cambia textos en `app.js`.
- Cambia colores en `styles.css`, dentro de `:root`.
- Cambia nombre o icono en `manifest.webmanifest`.
